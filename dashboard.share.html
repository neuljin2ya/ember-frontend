Ember DB 실시간 대시보드

유저를 클릭하면 해당 유저의 일기, 매칭, 교환일기, 채팅을 한눈에 볼 수 있습니다. 5초마다 자동 갱신.

📖 사용 가이드 (클릭하여 펼치기)
▸ 빠른 시작 (5단계)
1. [유저 + 온보딩 완료] 클릭 → 신규 유저 + 약관·프로필·이상형까지 자동 생성
2. 좌측 유저 목록에서 방금 만든 유저를 클릭해 선택
3. [시드 일기 작성] 영역에서 날짜·내용 입력 → [일기 추가] (AI 분석 5~10초 후 자동 부착)
4. 다른 유저도 만들고 일기 추가 → [A→B 매칭 신청] → [B가 수락] → 교환방 자동 생성
5. [4턴 연속 작성] → [A: 채팅 선택] + [B: 채팅 선택] → 채팅방 생성
▸ 버튼 색상
초록 = 생성/긍정 · 빨강 = 삭제 · 파랑 = 매칭/탐색 · 주황 = 일기 · 핑크 = 커플 · 노랑 = 강제/디버그
▸ 자동 갱신
DB 상태는 5초마다 자동으로 새로 가져옵니다. 즉시 반영하려면 [새로고침] 버튼.
▸ Tip
버튼에 마우스를 올리면 자세한 설명이 표시됩니다.
