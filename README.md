# Ember

교환일기 기반 AI 소개팅 앱 'Ember'

## 프로젝트 소개

기존 소개팅 앱은 사진과 짧은 프로필 중심으로 상대를 판단하게 되어 피상적인 관계 형성에 그치는 경우가 많습니다.

Ember는 사용자의 일기와 감정, 관심사를 기반으로 상대를 추천하고 교환일기를 통해 자연스럽게 관계를 형성할 수 있도록 기획된 서비스입니다.

사용자는 AI 분석 결과를 통해 자신의 성향을 확인할 수 있으며, 매칭 이후에도 교환일기와 채팅을 통해 상대를 천천히 알아갈 수 있습니다.

단순 스와이프 중심의 소개팅 서비스가 아닌, 기록과 소통을 기반으로 관계를 만들어가는 새로운 소개팅 경험을 제공하는 것을 목표로 합니다.
